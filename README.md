


# AI Examiner Agent — NLP Course

## Опис системи

Даний проєкт представляє собою інтелектуальну систему для автоматизованого проведення усного іспиту з курсу *Natural Language Processing (NLP)*.
Система реалізована у вигляді вебзастосунку з чат-інтерфейсом, де студент взаємодіє з AI-екзаменатором, який послідовно проводить іспит відповідно до заданих правил.

AI-екзаменатор:

* обирає кілька тем з курсу NLP;
* ставить базові та поглиблені запитання з кожної теми;
* контролює кількість запитань і порядок переходу між темами;
* автоматично формує оцінку та текстовий відгук після завершення іспиту.

Користувацький інтерфейс реалізований за допомогою Gradio, а генерація відповідей відбувається через Groq API з використанням великої мовної моделі Llama 3.3 70B.



## Задача, яку розв’язували

Основною задачею було створення системи, яка:

* автоматизує процес перевірки знань студентів з NLP;
* імітує формат усного іспиту;
* забезпечує структурований та справедливий підхід до оцінювання;
* може бути легко розгорнута як вебсервіс.

## Інструкції з запуску

### Локальний запуск

1. Клонувати репозиторій:

```bash
git clone https://github.com/eleonora31krech/agent_nlp_lab.git
cd agent_nlp_lab
```

2. Встановити залежності:

```bash
pip install -r requirements.txt
```

3. Запустити застосунок:

```bash
python app.py
```

4. Відкрити у браузері:

```
http://localhost:7860
```



### Особливості запуску

Для роботи застосунку необхідний API ключ Groq.
Ключ можна:

* ввести безпосередньо в інтерфейсі, або
* зберегти як змінну середовища:

```bash
export GROQ_API_KEY=gsk_...
```
